## Usage Examples

You can use these components in any MDX file like this:

```mdx
import { ToolComparison, FeatureMatrix, WakaTimeChart } from '@/components/mdx'

# My Article

<ToolComparison tools={...} />
```

## Conclusion

These 7 components provide a rich set of interactive features for your blog:

1. **ToolComparison** - Side-by-side pros/cons with ratings
2. **FeatureMatrix** - Sortable tables with color-coded scores
3. **WakaTimeChart** - Beautiful data visualizations
4. **ToolTimeline** - Horizontal/vertical event timelines
5. **WorkflowDiagram** - Circular and linear process flows
6. **VersionDiff** - Git-style code differences
7. **ToolList** - Searchable tool directories

Try searching or filtering the ToolList above, click the table headers in FeatureMatrix to sort, or hover over the timeline events for more details!

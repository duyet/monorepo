![Cowork as a vision planner](/media/2026/01/ai/cowork-vision-planner.png)

I use Cowork for a lot more than Claude Code. It works as a vision planner and a market research tool when I'm working on a project. I let it think through the product direction and vision, then break the work into nightly tasks for Claude Code to run overnight. It keeps track of all the progress, so the product stays on track.

![Cowork handing the plan to Claude Code](/media/2026/01/ai/cowork-claude-code-planner.png)

I just check the progress through this one big execution plan. PRs get auto-merged, I preview the result, and cut a release when I feel good about it.
